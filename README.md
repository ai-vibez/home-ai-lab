# Home AI Lab

Private local AI inference stack. Ubuntu 26.04, GTX 980 Ti, llama.cpp + Ollama + Open WebUI.

## Stack
- llama-server (GPU inference, port 8081)
- Ollama (model management, port 11434)
- Open WebUI (interface, port 3000)
- Qdrant (vector DB, coming soon)
- n8n (automation, coming soon)

## Goal
Telecom AI applications. RAG pipelines. Income independence.
